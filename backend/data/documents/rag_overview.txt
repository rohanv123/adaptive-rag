Retrieval-Augmented Generation (RAG) is a technique that combines information retrieval with text generation.

Instead of relying only on a language model’s internal knowledge, RAG retrieves relevant documents from a knowledge base and uses them as context for generating answers.

The pipeline consists of:
1. Query input
2. Retrieval of relevant documents
3. Passing retrieved context to a language model
4. Generating a grounded answer

RAG improves factual accuracy and reduces hallucinations by grounding responses in real data.
